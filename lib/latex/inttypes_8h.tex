\subsection{Detailed Description}
Provides A\+PI for format conversion of integer types. 

It declares four functions for converting numeric character strings to greatest-\/width integers and, for each type declared in $<$\mbox{\hyperlink{stdint_8h}{stdint.\+h}}$>$, it defines corresponding macros for conversion specifiers for use with the formatted input/output functions.

These macros are provided mainly in support of fprintf and fscanf.
