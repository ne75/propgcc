Start a cog with a parameter. 

The fields in parameters are\+:

\begin{DoxyItemize}
\item 31\+:18 = 14-\/bit Long address for P\+AR Register \item 17\+:4 = 14-\/bit Long address of C\+O\+DE to load \item 3 = New bit \item 2\+:0 = Cog ID. New bit 3 is 0.\end{DoxyItemize}
It is important to realize that a 14 bit address means that long aligned addresses or pointers should be use. That is if you pass a value to the P\+AR such as 3, the value will be truncated to 0. A value 5 will be interpreted as 4.
