\subsection{Detailed Description}
Provides complex math A\+PI declarations. 

Complex numbers are numbers of the form z = a+b$\ast$i, where a and b are real numbers and i = sqrt(-\/1), so that i$\ast$i = -\/1.

There are other ways to represent that number. The pair (a,b) of real numbers may be viewed as a point in the plane, given by X-\/ and Y-\/coordinates.

This same point may also be described by giving the pair of real numbers (r,phi), where r is the distance to the origin O, and phi the angle between the X-\/axis and the line Oz. That is\+: z = r$\ast$exp(i$\ast$phi) = r$\ast$(cos(phi)+i$\ast$sin(phi)).

The basic operations are defined on z = a+b$\ast$i and w = c+d$\ast$i as\+: \begin{DoxyItemize}
\item addition\+: z+w = (a+c) + (b+d)$\ast$i \item multiplication\+: z$\ast$w = (a$\ast$c -\/ b$\ast$d) + (a$\ast$d + b$\ast$c)$\ast$i \item division\+: z/w = ((a$\ast$c + b$\ast$d)/(c$\ast$c + d$\ast$d)) + ((b$\ast$c -\/ a$\ast$d)/(c$\ast$c + d$\ast$d))$\ast$i\end{DoxyItemize}
Nearly all math function have a complex counterpart but there are some complex-\/only functions.

Your C-\/compiler can work with complex numbers if it supports the C99 standard. Link with -\/lm. The imaginary unit is represented by I.

Most detailed descriptions of these functions come from B\+SD man pages. See file lib/\+L\+I\+B\+\_\+\+L\+I\+C\+E\+N\+S\+E.\+txt for license info.

Complex hyperbolic tangent function. 

ctanh(z) computes the hyperbolic tangent of the complex floating-\/point number z.

For all complex floating point numbers z,

\begin{DoxyItemize}
\item ctanh(conj(z)) = conj(ctanh(z)), \item ctanh(-\/z) = -\/ctanh(z).\end{DoxyItemize}
The symmetries of \mbox{\hyperlink{complex_8h_a9c5cf378e654f3aaed9fb0f5c8223585}{ctanh()}} are used to abbreviate the specification of special values.

\begin{DoxyItemize}
\item ctanh(0 + 0i) returns 0 + 0i. \item ctanh(0 + inf i) returns NaN + NaN i, and raises the invalid flag. \item ctanh(0 + NaN i) returns NaN + NaN i. \item ctanh(x + inf i) returns NaN + NaN i, and raises the invalid flag, for finite nonzero x. \item ctanh(x + NaN i) returns NaN + NaN i, for finite nonzero x. \item ctanh(inf + 0i) returns 1 + 0i. \item ctanh(inf + yi) returns 1 +-\/ 0i, for finite positive y, with the sign chosen to match the sign of sin(2y). \item ctanh(inf + inf i) returns 1 + 0i. \item ctanh(inf + NaN i) returns 1 + 0i. \item ctanh(NaN + 0i) returns NaN + 0i. \item ctanh(NaN + yi) returns NaN + NaN i, for nonzero numbers y. \item ctanh(NaN + NaN i) returns NaN + NaN i.\end{DoxyItemize}
