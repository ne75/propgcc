\subsection{Detailed Description}
Defines macros for limits and parameters of the standard integer types. 

The compiler supports data with the following characteristics\+: \begin{DoxyItemize}
\item 8 bit char \item 16 bit short \item 32 bit int \item 32 bit long \item 64 bit long long\end{DoxyItemize}
The compiler supports U\+T\+F-\/8, but does not support multi-\/byte characters.
