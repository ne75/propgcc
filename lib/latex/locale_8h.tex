\subsection{Detailed Description}
Provides multi-\/lingual localization A\+PI functions. 

C localization functions are used in multilingual programs to adapt to the specific locale for displaying numbers and currency. Functions affect the C Standard Library I/O function behavior.

The only locales supported is \char`\"{}\+C\char`\"{} and a C.\+U\+TF-\/8 locale which is the same as C but supports U\+T\+F-\/8 multibyte character encoding. Things like currency and so forth are hard coded to the C locale.
