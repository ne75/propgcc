\subsection{Detailed Description}
Provides character class check A\+PI macros. 

These functions check whether c, which must have the value of an unsigned char or E\+OF, falls into a certain character class according to the current locale.

Conforms to C99, 4.\+3\+B\+SD. C89 specifies all of these functions except isascii() and \mbox{\hyperlink{ctype_8h_a8953d6a09200454cf6c0c8500441ca19}{isblank()}}. isascii() is a B\+SD extension and is also an S\+Vr4 extension. \mbox{\hyperlink{ctype_8h_a8953d6a09200454cf6c0c8500441ca19}{isblank()}} conforms to P\+O\+S\+I\+X.\+1-\/2001 and C99 7.\+4.\+1.\+3. P\+O\+S\+I\+X.\+1-\/2008 marks isascii() as obsolete, noting that it cannot be used portably in a localized application.

\subsubsection{\texorpdfstring{tolower()}{tolower()}}
{\footnotesize\ttfamily int tolower (\begin{DoxyParamCaption}\item[{int}]{c }\end{DoxyParamCaption})}

Converts an uppercase letter to a corresponding lowercase letter.

If the argument is a character for which isupper is true and there are one or more corresponding characters, as specified by the current locale, for which islower is true, the tolower function returns one of the corresponding characters (always the same one for any given locale); otherwise, the argument is returned unchanged.
